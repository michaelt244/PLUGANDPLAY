API Routes

POST /api/generate-ad — multipart photo + business context → Claude Vision → 3 variants
POST /api/dispatch — variant + platforms → Manus → real posts to FB / IG / Google Business

Tech Stack

Frontend

Next.js 14 App Router · Tailwind CSS · shadcn/ui

AI

Claude Vision (Anthropic SDK) for ad copy generation

Dispatch

Manus for automated posting to FB, IG, Google Business

Database

Supabase — campaigns table stores ad variants + dispatch results

Supabase Schema

campaigns (
  id uuid,
  business_name text,
  ad_goal text,
  tone text,
  photo_url text,
  variants jsonb,    -- [{label, copy, platform}]
  selected_variant int,
  platforms text[],   -- ['facebook','instagram','google']
  dispatch_status jsonb, -- {facebook: 'posted', ...}
  created_at timestamptz
)
